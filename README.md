Bit = binary + digit

비트는 2진법을 사용한다는 뜻의 binary와 숫자를 의미하는 digit이 합쳐진 말이다

불리언대수: 영국 수학자 조지 불이 만들어낸 연산 규칙의 집합
NOT: 논리적 반대를 의미하며 거짓인 비트에 NOT을 하면 참이되고 참인 비트에 NOT을 하면 거짓이 된다

AND: 둘 이상의 비트에 사용되며 계산하는 모든 비트가 참인 경우에만 결과가 참이 된다

OR: 둘 이상의 비트에 사용되며 계산하는 모든 비트중 하나만 참이더라도 결과가 참이된다

XOR: exclusive OR, 배타적 OR이며 첫번째 비트와 두번째 비트가 다른 경우에만 참이된다, 두 비트가 모두 참이면 결과는 거짓이고 두 비트중
하나만 참인 경우 결과가 참이된다. 인터넷 검색시 3개이상의 계산도 가능하며 참인 갯수가 즉 1의 갯수가 홀수인 경우 참이라고 한다.

드모르간의 법칙: 영국 수학자인 오거스터스 드모르간이 고안한 불리언대수에 적용할수 있는 추가법칙

A AND B = NOT(NOT A OR NOT B)
다음과 같이 불리언대수를 조합하여 다른 식이지만 같은 결과를 만들어 낼수있다. NOT연산을 수행하는 하드웨어가 돈이 더 들어가며
연산을 연쇄적으로 사용하면 계산이 느려진다는 점에서 드모르간의 법칙을 알고 활용하는것이 중요한것 같다

MSB: most significant bit로 가장 큰 유효비트이다, 가장 왼쪽의 비트를 가리키며 가장 왼쪽의 비트를 변경하면 2진수의 값이 가장 크게 변해서 이러한 이름이 붙게되었다

LSB: least significant bit로 가장 작은 유효비트이다, 가장 오른쪽의 비트를 가리키며 가장 오른쪽의 비트를 변경하면 2진수의 값이 가장 작게 변경된다

음수표현방법
1. 부호와 크기

MSB를 부호표현을 하기위해 사용하는 방법이다, MSB가 0이면 2진수를 양수로 취급하고 MSB가 1이면 음수로 취급하는방법
부호와 크기방법은 두가지 단점이있는데 이로인해 잘 사용되지 않고있다

1-1. 0을 표현하는 방법이 두가지라 비용이 낭비된다
2진수로 표현된 1000과 0000이 모두 각각 -0 과 +0을 표기하므로 비용이 낭비됨

1-2. XOR과 AND를 통한 덧셈 계산을 사용할수없다
+1을 의미하는 2진수 0001과 -1을 의미하는 2진수 1001를 계산하면 1010이며 이는 부호와크기 표현법에서 -2를 의미한다, 실제로는 두 정수의 계산은 0이기 때문에
잘못된 결과가 나오게됨

2. 1의보수

양수의 모든 비트를 뒤집는 방식으로 음수를 표현하는방법이다
부호와 크기 표현법과 마찬가지로 MSB를 부호표기로 사용한다, 예를들어 부호와크기 표현법에서 0111을 10진수로 +7을 표현한다는 점까지는 같지만
그 이후 0111의 모든 비트를 뒤집은 1000이 1의보수 표현법에서는 -7이된다
1의보수도 0을 두가지방식으로 표현한다는 비용적인 문제가 있으며 덧셈 계산간 올림수를 처리하는 하드웨어를 추가해야 한다는 문제로 잘 사용되지 않는다

3. 2의보수

어떤 양의정수를 비트를 뒤집고 거기에 1을 추가한것을 음수로 표기하기로 약속한 방식이다
예를들어 0001은 +1을 의미하며 해당 비트를 뒤집은것은 1110이고 여기에 1을더한 1111을 -1로 표현한다
0을 표현하는 방법이 하나이며 가장 널리사용되는 방식이다, 4비트의 경우 표현할수있는 값의 법위가 +7 ~ -8 이라는 점을 유의할 필요가있다

실수표현방법

1. 고정소수점 표현법

2진수에서 소수점의 위치를 임의로 정하는방법이다, 예를들어 4비트중 2비트는 정수를 표현하는데에 쓰고 2비트는 실수를 표현하는데에 쓰는 방식으로
0101에서 왼쪽의 두 비트 01은 정수 1, 오른쪽의 두 비트 01은 4분의1을 나타내 최종값은 1.25가 된다

고정소수점 표현방식은 잘 작동하기는 하지만 다양한값의 실수를 표현하기 위해서는 필요한 비트수가 너무 많기때문에 범용컴퓨터에서는 잘 사용되지 않으며
특수한 목적에따라 일부 사용되기는 한다

2. 부동소수점 표현법

과학적 표기법을 사용하여 가수부분과 지수부분을 나누어 실수를 표현하는 방식이다
예를들어 10진수에서 과학적 표기법을 사용해 1200을 나타내면 1.2 x 10의3승이 된다 여기서 1.2는 가수이고 10의3승은 지수가 된다.

4비트에서 부동소수점 표현법을 사용하면 왼쪽 두 비트는 가수이고 오른쪽 두 비트는 지수가되고 1111은 1.5 x 2의3승으로 12가된다
